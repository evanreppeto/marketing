# Arc Redesign — Phase 0 (Foundation) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the remaining shared foundation the 20 premium mockups need — warm-obsidian canvas, an app-wide top bar with ⌘K search + avatar, and the two missing primitives (`EvidenceChip`, `InlineApprovalCard`) — so every later page cycle inherits the look.

**Architecture:** Extend the shipped #259 Foundation Pass (calm flat canvas, hairline panels, Work/Studio/Intelligence IA). Pure presentation + two new presentational components; no backend, no data, no route logic. All color comes from CSS variables in `globals.css` (no hex in components), so recoloring is a token edit. New primitives are server-renderable (no hooks) and consume `theme.ts` tokens.

**Tech Stack:** Next.js 16 (App Router), React 19, Tailwind v4 (`@import "tailwindcss"`), CSS custom properties, TypeScript, vitest (node env — no jsdom/RTL, so UI verifies via `tsc` + browser preview), pnpm.

**Spec:** [`docs/superpowers/specs/2026-06-24-arc-redesign-design.md`](../specs/2026-06-24-arc-redesign-design.md). **Look reference:** mockups at `localhost:8910` (`%TEMP%\arc-ui-mockups`), especially `01-command-center.png`.

**Conventions (from CLAUDE.md / DESIGN.md / memory):**
- `pnpm lint` scans vendored files (~31k problems) — always scope eslint to changed files: `npx eslint <path>`.
- `pnpm lint` does NOT typecheck — use `npx tsc --noEmit` to catch type errors.
- Build from `globals.css` `:root` tokens + `theme.ts`; never hard-code hex or one-off class maps.
- There is no `--surface` token — use `--canvas` / `--surface-panel` / `--surface-inset` / `--surface-raised`.
- Commit after each task.

## Task 4: Add the `EvidenceChip` primitive

A small source/citation chip used across Opportunities, the Home focal card, Arc replies, and Brand facts. Server-renderable (no hooks).

**Files:**
- Create: `src/app/_components/evidence-chip.tsx`

- [ ] **Step 1: Create the component**

Create `src/app/_components/evidence-chip.tsx` with exactly:

```tsx
import Link from "next/link";

import { cx } from "./theme";

/**
 * Small source / evidence chip — the citation cue used across Opportunities,
 * the Home focal card, Arc replies, and Brand facts. Optional leading index
 * (rendered as [n]) and an optional confidence read; links out when `href` is
 * set. Presentational and server-renderable (no hooks).
 */
export function EvidenceChip({
  index,
  label,
  confidence,
  href,
  className = "",
}: {
  /** Optional citation number, rendered as [n]. */
  index?: number;
  /** Source name, e.g. "NOAA" or "TechCrunch". */
  label: string;
  /** Optional 0–1 confidence; rendered as a percentage. */
  confidence?: number;
  href?: string;
  className?: string;
}) {
  const inner = (
    <>
      {typeof index === "number" ? (
        <span className="font-mono text-[10px] text-[var(--accent)]">[{index}]</span>
      ) : null}
      <span className="truncate">{label}</span>
      {typeof confidence === "number" ? (
        <span className="font-mono text-[10px] tabular-nums text-[var(--text-muted)]">
          {Math.round(confidence * 100)}%
        </span>
      ) : null}
    </>
  );

  const classes = cx(
    "inline-flex items-center gap-1.5 rounded-[6px] border border-[var(--border-hairline)] bg-[var(--surface-inset)] px-2 py-0.5 text-[11px] font-medium text-[var(--text-secondary)] transition",
    href ? "hover:border-[var(--accent-border)] hover:text-[var(--text-primary)]" : "",
    className,
  );

  if (href) {
    const external = href.startsWith("http");
    return (
      <Link
        href={href}
        className={classes}
        target={external ? "_blank" : undefined}
        rel={external ? "noopener noreferrer" : undefined}
      >
        {inner}
      </Link>
    );
  }
  return <span className={classes}>{inner}</span>;
}
```

/**
